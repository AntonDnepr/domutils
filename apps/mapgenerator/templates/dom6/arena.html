Welcome to the Arena Map Tool

This tools makes extensive usage of Arena Map by Dastictic. You don't need to install it, but you can do so here

Please support the author of the original map.

You can help the development of this tool or report a bug here.

Tool Steps:

  1. Select nations that will be used in map generation.

    There is no validation that nations are selected from the same Age. You should select at least 2 nations from the same Age for the game to not declare you a winner immediately.
    There is no validation for land / water nation, so please be careful when selecting them.

  2. One by one, you will be prompted to generate commanders and units for the selected nations.

    You can skip nations if you do not want them having anything but a starting expansion party.
