Welcome to the Arena Map Tool

This tool makes extensive use of Arena Map by Dastactic. Please support the author of the original map.

You can contribute to the development of this tool or report a bug here.

You can check tutorial here or read short description below.

Tool Steps:

  1. Select the nations to be used in map generation.

    There is no validation that nations are selected from the same Age. You should select at least 2 nations from the same Age for the game to not declare you the winner immediately.
    There is no validation for land / water nation, so please be careful when selecting them.

  2. One by one, you will be asked to create commanders and units for the selected nations.

    You can skip nations if you do not want them to have anything other than an initial expansion party.

  3. At the end you will get a zip archive with the map. You should unzip it into the map folder of the game.

    Map folder is available by clicking "Tools & Manuals" -> "Open User Data directory"
